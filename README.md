# House Price Prediction and Classification using SVM

## Overview
This project predicts house prices using Support Vector Regression (SVR) and classifies houses into Low, Mid, and High price categories using Support Vector Classification (SVC).

## Dataset
King County House Sales Dataset (Seattle, WA)
Dataset available from Kaggle:
https://www.kaggle.com/datasets/harlfoxem/housesalesprediction

## Techniques Used
- Data Preprocessing
- StandardScaler
- Train-Test Split
- Support Vector Regression (SVR)
- Support Vector Classification (SVC)
- SMOTE
- Learning Curves
- Confusion Matrix

## Results
- House Price Prediction
- Price Category Classification
- Class Imbalance Handling using SMOTE
